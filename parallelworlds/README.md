# ParallelWorlds

Performant custom blocks without client modification


## Introduction

## Features

- Create deeply integrated, custom blocks with ease 
- Simple programming API to add even more advanced block functionality 
- Automatically generate resource packs
- Seamlessly display visual assets to clients with packet injection
- Merge resource packs with provided packs or other plugins for ease of use
- Directly serve resource packs from server and force clients to use

## Compatibility 

This aims to be fully compatible with mods by deeply integrating with the Paper server.
Due to the nature of these integrations, some bugs may still exist!
Please report all compatibility issues on the Issues tab on GitHub.
